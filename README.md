## Overview

This project investigates patterns of horizontal gene transfer (HGT) using genomic feature analysis and machine learning methods. The goal is to identify sequence-level and compositional signals that may indicate gene transfer events across microbial genomes.

The pipeline combines biological sequence processing with statistical and machine learning approaches to classify or detect potential gene transfer regions.

This project was developed as part of my independent exploration in bioinformatics and computational genomics.
## Objectives

- Extract biologically relevant features from genomic sequences
- Identify patterns associated with potential horizontal gene transfer
- Apply machine learning models to classify gene transfer signals
- Visualise results using publication-style scientific plots

## Data Sources

- Genomic sequence data from [NCBI / GTDB]
- Annotated gene regions used for feature extraction
- Processed using Python-based bioinformatics pipeline
